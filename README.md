# Movie Recommendation System using Graph Neural Networks

This project implements a movie recommendation system using a graph neural network (GNN) trained on user-movie interaction data.

## Dataset
- users.csv
- movies_fixed.csv
- ratings.csv (hosted externally due to large size)

Ratings dataset link:
<https://drive.google.com/file/d/1dgZ5t6jceU-KqlHn9SAIFpFVvL8aH3FV/view?usp=drive_link>

## Tech Stack
- Python
- PyTorch
- PyTorch Geometric
- Pandas
- Scikit-learn
- Jupyter Notebook

## How to Run
1. Install dependencies
2. Download ratings.csv from the link
3. Place it in the project folder
4. Run the notebook
